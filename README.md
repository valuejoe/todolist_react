# 運用react及redux來寫簡易的todolist

### 基本功能
-   Add Todo
-   Delete Todo
-   Edit Todo
-   Complete Todo
-   Star Todo
-   Filter
### 運用工具
-   React : 主要使用的前端框架
-    Redux : 資料傳輸使用的API
-   react-sortable-hoc : 用來做todo的拖曳及排序的API
-   CSS : 呈現網頁樣貌
-   webpack :  主要開發環境，用來整合及管理網頁元件
